# Chess

A implementation of chess using Python and Pygame.

## Installation

1. Install UV:

```bash
# On macOS and Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# On Windows
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

2. Run the project:

```bash
# Install dependencies and run
uv run main.py
```
